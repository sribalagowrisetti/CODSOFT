💼 Job Board Website

A simple and responsive Job Board Web Application built using HTML, CSS, and JavaScript as part of the CodSoft Internship project.

🚀 Features
🔍 Search jobs by title or company
💼 Display job listings in card layout
📱 Responsive design (works on mobile & desktop)
⚡ Simple and clean UI
🖱️ "Apply Now" button UI (frontend only)
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
