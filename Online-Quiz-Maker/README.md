# Online Quiz Maker

## Description

The Online Quiz Maker is a simple web application developed using HTML, CSS, and JavaScript. It allows users to answer multiple-choice questions and view their final score.

## Features

* Multiple-choice questions
* Score calculation
* Interactive user interface
* Responsive design
* Instant result display

## Technologies Used

* HTML
* CSS
* JavaScript
